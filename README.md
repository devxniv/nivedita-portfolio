# Nivedita Bedarkar — Portfolio

A personal portfolio website built with React, Vite, and Tailwind CSS.

🌐 Live: [View Portfolio](https://niveditaportfolio.vercel.app)

---

## About

This is my personal portfolio showcasing my projects, skills, education, and certifications. Built from scratch as part of my journey as a full-stack developer.

---

## Tech Stack

| Technology      | Purpose      |
| --------------- | ------------ |
| React 19        | UI framework |
| Vite            | Build tool   |
| Tailwind CSS v4 | Styling      |
| Framer Motion   | Animations   |
| React Icons     | Icons        |
| EmailJS         | Contact form |

---

## Features

- Responsive design — works on mobile and desktop
- Animated sections with Framer Motion
- Functional contact form via EmailJS
- Projects showcase with live demo and code links
- Education timeline
- Certifications section

---

## Getting Started

```bash
git clone https://github.com/devxniv/nivedita-portfolio.git
cd nivedita-portfolio
npm install
npm run dev
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
